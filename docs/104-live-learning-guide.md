# Live Learning Guide

Live sessions support scheduling, waiting-room state, hosts, participants, attendance, chat, Q&A, polls, recordings, raise-hand state, and post-session metadata. Zoom, Google Meet, Microsoft Teams, and BigBlueButton are provider labels only; no SDK or credential is present.

Mentors publish office-hour availability and learners book, reschedule, or cancel through controlled workflows. Study groups own member and session metadata. Whiteboards preserve snapshot references, hashes, versions, and export references without implementing a drawing surface.

Attendance and recording evidence is immutable. Join and attendance RPCs verify active tenant membership and host authorization. UI join controls remain unavailable when a provider is unconfigured or a session is not live.
