# Assessment Authoring Guide

## Scope

Assessment authoring provides blueprints, templates, sections, question ordering, randomization metadata, pools, review assignments, and publishing workflow.

## Runtime Integration

Assessment templates reference existing runtime assessment entities when published. The authoring system does not modify learner assessment pages, active attempts, submissions, answer-key handling, or scoring runtime.

## Workflow

The approved workflow is draft, review, approved, scheduled, published, and archived. Rejected content returns to the editable draft path.

## Evidence

Assessment authoring events and question change logs are append-only. Published question history is immutable.
