# Enterprise Reporting Guide

Reporting is organization-scoped. Administrators can define reports, run them, schedule them, export CSV, Excel, JSON or PDF metadata, and save dashboard layouts through controlled server actions. Mentors receive read-only projections.

No report contains private learner profile data by default. New projections must reference the contract, carry a tenant filter, and be added through a forward migration. The reporting runtime does not render PDF files; pdf_metadata is an export manifest only.
