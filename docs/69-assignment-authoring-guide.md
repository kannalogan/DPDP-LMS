# Assignment Authoring Guide

Authorized course authors, mentors and administrators create a stable assignment identity, save versioned drafts, submit versions for review, approve or reject them, and publish immutable versions. Assignment versions carry submission constraints, outcome metadata, due and late policies, resubmission policy, grading mode and a pinned rubric version.

Course, module and lesson associations are explicit junction records. Publication writes use controlled RPCs and create append-only publication and authoring evidence.
